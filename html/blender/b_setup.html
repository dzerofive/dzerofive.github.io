Скачать Blender для всех ОС можно на сайте Blender.org/Download/, или при помощи Package Manager-а ОС:

Ubuntu sudo apt-get install blender
Arch/Manjaro sudo pacman -S blender
Snapcraft sudo snap install blender

Для тех, кто решил через Package Manager, держу в курсе что некоторые плагины нужно кидать в папку с Блендером, тоесть если она где-то защищённой папке Package Manager-а, то с этим может быть лишняя боль.

Если у тебя на клавиатуре нет нумпада, смотри здесь.
