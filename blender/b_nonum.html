Эмуляция нумпада в Blender

<- назад
tldr: Edit - Preferences - Input - Emulate Numpad

Управление Блендера по умолчанию ожидает наличие Нумпада на клавиатуре, но если его нет, в настройках можно включить эмуляцию нумпада. Направляйся в Edit - Preferences - Input и поставь галочку на Emulate Numpad. Теперь клавиши 1-9, 0 над буквами будут делать то же, что и клавиши нумпада тех же цифр.
